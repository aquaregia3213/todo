## **AI Life OS MVP** 

## **Goal** 

Help students create and follow career roadmaps. 

## **Feature 1: User Profile** 

Collect: 

- Name • Age • Degree • Branch • Current Year • Skills • Interests • Career Goal • Budget • Weekly Free Time 

Store in database. 

## **Feature 1.5: User Authentication**

Secure access via:

- Email and Password Sign Up / Login

- Google and GitHub OAuth integrations

- Client-side JWT verification to restrict and personalize database entries per-user.

## **Feature 2: AI Roadmap Generator** 

Input: 

User Profile 

Output: 

## **30 Day Plan** 

- Skills to learn 

- Resources 

- Projects 

## **90 Day Plan** 

- Intermediate goals 

- Portfolio building

- Internship preparation
